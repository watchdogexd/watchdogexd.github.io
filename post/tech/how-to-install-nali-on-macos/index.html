在 macOS 上快速安装与配置 nali

介绍如何在 macOS 上安装 nali，实现终端 IP 信息自动解析。

推荐版本： Nxtrace/nali
该分支支持 NextTrace 的 LEOMOEAPI，数据更新更及时。

引言#

nali 是一款离线查询 IP 地理信息和 CDN 服务商的终端工具。它可以无缝衔接 ping, traceroute, curl 等常用命令。通过管道符（Pipe），它可以直接在终端输出中解析 IP 信息，省去了手动复制 IP 到网页查询的碎片操作。

安装步骤#

1. 下载二进制文件#

访问 nali Releases，根据你的芯片架构选择对应版本：

  • Apple Silicon: nali-darwin-arm64-v*.gz
  • Intel 处理器: nali-darwin-amd64-v*.gz

2. 命令行安装#

下载完成后，推荐通过终端快速解压并移动到系统路径：

# 解压文件（以 arm64 为例）
gunzip nali-darwin-arm64-v*.gz

# 赋予执行权限
chmod +x nali-darwin-arm64-v*

# 移动到 bin 目录并重命名为 nali
sudo mv nali-darwin-arm64-v* /usr/local/bin/nali
