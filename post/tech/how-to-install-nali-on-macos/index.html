在 macOS 上安装 nali

January 27, 2026 · 1 分钟 · 139 字 · kyorisu
目录
  • 引言
  • 安装

推荐使用 https://nxtrace.github.io/nali/
支持 NextTrace 的 LEOMOEAPI

引言#

nali 作为查询 IP 地理信息和 CDN 服务商的终端工具，非常适合日常使用 ping, traceroute, curl 等终端命令时，减少复制粘贴 IP 或域名到网页查询的碎片操作。

安装#

https://github.com/zu1k/nali/releases/

将下载好的 nali 二进制文件拷贝到 /usr/local/bin 即可在终端中使用 nali。
注意选择 nali-darwin-arm64-v*.gz (Apple Silicon)
或者 nali-darwin-amd64-v*.gz (Intel)

在终端中输入下列内容，确认 nali 是否已正确安装。

1

nali help
